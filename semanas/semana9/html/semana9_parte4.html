Ejercicio 1: Identifica el Sufijo Pronominal

¿Qué persona y número expresa cada sufijo?

1. סוּסִי (susí)
Sufijo: ִי (Tipo 1, 1cs)
Traducción: "mi caballo"
Persona: 1ª persona común singular (yo)
2. סוּסְךָ (susjá)
Sufijo: ְךָ (Tipo 1, 2ms)
Traducción: "tu caballo"
Persona: 2ª persona masculino singular (tú)
3. סוּסֵנוּ (susénu)
Sufijo: ֵנוּ (Tipo 1, 1cp)
Traducción: "nuestro caballo"
Persona: 1ª persona común plural (nosotros)
4. סוּסָם (susám)
Sufijo: ָם (Tipo 1, 3mp)
Traducción: "su caballo (de ellos)"
Persona: 3ª persona masculino plural (ellos)

Ejercicio 2: Traduce al Español

1. תּוֹרָתִי
Traducción: "mi ley"
Análisis: תּוֹרָה (ley) + sufijo 1cs ִי
2. דְּבָרֶיךָ
Traducción: "tus palabras"
Análisis: דָּבָר (palabra) plural + sufijo 2ms
3. אֶרְצְכֶם
Traducción: "vuestra tierra"
Análisis: אֶרֶץ (tierra) + sufijo 2mp
4. לִבָּהּ
Traducción: "su corazón (de ella)"
Análisis: לֵב (corazón) + sufijo 3fs ָהּ con Mappiq

Ejercicio 3: Preposiciones con Sufijos

1. לִי
Preposición base: לְ (a, para)
Traducción: "a mí / para mí"
Sufijo: 1cs
2. עִמָּנוּ
Preposición base: עִם (con)
Traducción: "con nosotros"
Sufijo: 1cp (¡como en Emanuel!)
3. אֵלָיו
Preposición base: אֶל (hacia)
Traducción: "hacia él"
Sufijo: 3ms
4. בָּנוּ
Preposición base: בְּ (en, con)
Traducción: "en nosotros / con nosotros"
Sufijo: 1cp

Ejercicio 4: El Marcador אֵת con Sufijos

1. אֹתִי
Traducción: "a mí / me"
Uso: Objeto directo (me, a mí)
2. אֹתְךָ
Traducción: "a ti / te" (masculino)
Uso: Objeto directo
3. אֹתָם
Traducción: "a ellos / los"
Uso: Objeto directo masculino plural
4. אֶתְכֶם
Traducción: "a vosotros / os"
Uso: Objeto directo 2mp

Ejercicio 6: ¿Posesión u Objeto?

Identifica si el sufijo expresa posesión (en sustantivo) u objeto (en preposición).

1. בֵּיתִי
Posesión — "mi casa" (sufijo en sustantivo)
2. לְךָ
Objeto — "a ti / para ti" (sufijo en preposición)
3. אֹתוֹ
Objeto directo — "a él / lo" (marcador אֵת con sufijo)
4. עַמְּךָ
Posesión — "tu pueblo" (sufijo en sustantivo עַם)
